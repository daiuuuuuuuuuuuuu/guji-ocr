## 你需要准备什么

| 序号 | 准备项 | 说明 |
|:----:|--------|------|
| 1 | 一台 Windows 电脑 | 能上网就行 |
| 2 | 一个 **DeepSeek API Key** | 去 [DeepSeek 开放平台](https://platform.deepseek.com/) 注册 → API Keys → 创建 |
| 3 | 一个 **DashScope API Key** | 去 [阿里云百炼](https://bailian.console.aliyun.com/) 注册 → 开通 DashScope → 创建 API Key |
| 4 | 你的古籍 PDF 或图片 | 放在一个文件夹里备用 |

> **两把 Key 的区别**：DashScope 用来 OCR（看图识字），DeepSeek 用来页序重排和错别字纠正。都需要。
>
> 没错，**不需要装 Python，不需要装依赖**。全部交给 AI。

## 第一步：下载安装 VS Code

1. 打开浏览器，访问 **https://code.visualstudio.com/**
2. 点那个大大的蓝色按钮 **Download for Windows**
3. 下载完双击安装文件，一路点 **下一步**，全部默认就行
4. 装完后打开 VS Code，你应该看到这样的界面：

![VS Code 界面示意 — 左侧有文件图标、搜索图标等几个按钮的深色窗口]

---

## 第二步：安装 Claude Code 插件

1. 在 VS Code 左侧，点那个**四个方块**的图标（扩展商店，也可以按 `Ctrl+Shift+X`）
2. 在搜索框输入 **Claude Code**
3. 找到由 Anthropic 官方发布的那个，点 **安装**
4. 装完后左侧会出现一个新的 Claude 图标

---

## 第三步：下载安装 CC Switch

> CC Switch 是一个**桌面软件**（不是 VS Code 插件），用来让 Claude Code 走你便宜的 DeepSeek Key，而不是 Anthropic 官方 API。开源免费。

1. 打开浏览器，访问 **https://github.com/szh1118/cc-switch/releases**
2. 在最新版本下面找到 Windows 安装包（`.msi` 后缀的那个）
3. 下载后双击安装，一路点 **下一步** 就行
4. 安装完打开 CC Switch，它会出现在屏幕**右下角系统托盘**（小图标区域）

> 如果 GitHub 下载慢，去官网 **ccswitch.io** 找国内下载。

---

## 第四步：配置 CC Switch

### 4.1 跳过 Claude 登录验证

1. 在系统托盘右键点 CC Switch 图标 → **设置**
2. 找到 **"跳过 Claude 登录验证"** 开关，打开它

### 4.2 配置 DeepSeek API

1. 在 CC Switch 设置里找到 **API Provider / 供应商**
2. 选 **DeepSeek**，填入你的 DeepSeek API Key
3. 模型选 **deepseek-v4-flash**
4. 保存

> 配置完成。现在 Claude Code 会走你的 DeepSeek Key，一个项目几分钱。

## 第五步：配置项目的 API Key（1 分钟）

> CC Switch 的 Key 是给 Claude Code 用的，这里的 Key 是给古籍识别项目本身用的。**两套 Key 不冲突，需要分别配置。**

### 方式一：用 .env 文件（推荐，最简单）

1. 在项目文件夹里找到 `.env.example` 文件
2. 右键 → 复制，然后把副本**改名**为 `.env`（就去掉 `.example` 后缀）
3. 用记事本打开 `.env`，把里面的 Key 换成你自己的：

```ini
# 换成你的 DashScope Key（去阿里云百炼申请）
GUJI_LLM_API_KEY=sk-你的DashScope的Key

# 换成你的 DeepSeek Key（去 DeepSeek 开放平台申请）
GUJI_VERIFY_API_KEY=sk-你的DeepSeek的Key

# 换成你的 DeepSeek Key（可以和上面同一个）
GUJI_CORRECT_API_KEY=sk-你的DeepSeek的Key
```

4. 保存，搞定。

> 三个 Key 里，`GUJI_LLM_API_KEY` 必须是 **DashScope** 的，另外两个必须是 **DeepSeek** 的。别填反了。

### 方式二：在 VS Code 里直接改

如果你觉得在文件夹里改名麻烦，也可以在 VS Code 左边文件列表里，右键 `.env.example` → **复制** → 粘贴 → 命名为 `.env` → 打开改 Key → 保存。一样的。

### 方式三：让 Claude 帮你改

如果你连改文件都不想，到下一步直接跟 Claude 说：

```
项目里有个 .env.example 文件，帮我把它复制成 .env，
然后把里面的 Key 填好。我的 DashScope Key 是 xxx，
DeepSeek Key 是 xxx。
```

Claude 会自己处理好。

## 第六步：打开项目，让 AI 接手

### 6.1 打开项目文件夹

1. 把本项目文件夹解压到你电脑上的任意位置（比如桌面）
2. 在 VS Code 中，点 **文件 → 打开文件夹**
3. 选中项目文件夹，点确定

### 5.2 打开 Claude Code

1. 点击左侧 Claude 图标，打开 Claude Code 面板
2. 在对话框里输入下面这段话，回车：

```
你好，请帮我把这个古籍识别项目跑起来。

首先检查一下我的电脑上有没有 Python，如果没有就帮我安装。
然后帮我安装这个项目的所有依赖。
然后读取 .env 文件，帮我看看 API Key 配好了没有，如果没配好请告诉我。
最后跑一遍测试确认一切正常。
```

3. Claude 会自动检测你的电脑环境，缺什么装什么。如果中途遇到问题，它会告诉你。

> 如果弹出权限请求，点 **允许** 就好。

### 5.3 等待完成

Claude 会一步步做：
1. 检查 Python → 没有就装
2. 创建虚拟环境 → 安装依赖
3. 检查 config.yaml → 告诉你要改什么
4. 跑测试 → 确认没问题

整个过程你只需要看着就行。

---

## 第七步：把古籍扔进去，开工

### 6.1 放入古籍文件

把你的 PDF 或图片文件放入项目里的 **`古籍图片/`** 文件夹（没有就建一个）。

### 6.2 告诉 Claude 开始识别

在 Claude Code 对话框输入：

```
古籍图片/ 目录下是我的古籍 PDF，请帮我跑完整的三步流水线：
1. OCR 识别
2. 页序重排
3. 错别字纠正

最终结果放到 out_c 目录。
```

Claude 会依次执行三条命令，每步完成都会告诉你进度。

### 6.3 查看结果

打开 `out_c/你的书名/txt/`，双击 `.txt` 文件就能看到识别出来的文字。

## 常见问题

<details>
<summary><b>Q: CC Switch 找不到下载地址？</b></summary>

→ 直接访问 https://github.com/szh1118/cc-switch/releases ，或者在浏览器搜 "cc-switch github"。如果 GitHub 打不开或下载慢，去官网 ccsitch.io 找国内下载。
</details>

<details>
<summary><b>Q: Claude Code 提示需要登录？</b></summary>

→ 说明 CC Switch 的"跳过登录"没有生效。检查：
1. CC Switch 是否正确安装
2. 跳过登录开关是否打开
3. 重启 VS Code 再试
</details>

<details>
<summary><b>Q: 为什么要两把 Key？CC Switch 不是已经配过 DeepSeek 了吗？</b></summary>

→ 两套 Key 是给不同东西用的：
- CC Switch 的 Key → 给 **Claude Code 本身**用（让它能跟你对话）
- .env 里的 Key → 给**古籍识别项目**用（OCR + 页序重排 + 错别字纠正）

古籍识别需要**两个不同的 AI 服务**：OCR 用阿里云的 DashScope，页序和纠错用 DeepSeek。所以 .env 里要填两把 Key，CC Switch 只填 DeepSeek。
</details>

<details>
<summary><b>Q: DeepSeek Key 怎么获取？</b></summary>

→ 访问 https://platform.deepseek.com/ ，注册账号后在 API Keys 页面创建。新用户通常有免费额度。
</details>

<details>
<summary><b>Q: DashScope Key 怎么获取？</b></summary>

→ 访问 https://bailian.console.aliyun.com/ ，注册 → 开通 DashScope 服务 → API Keys → 创建。新用户有免费额度。
</details>

<details>
<summary><b>Q: Claude 执行命令报错？</b></summary>

→ 把错误信息复制给 Claude，让它自己排查。大多数情况下它会自动修复。
</details>

<details>
<summary><b>Q: 要钱吗？</b></summary>

→
| 项目 | 费用 |

| VS Code | 免费 |

DeepSeek 新用户注册就送免费额度，一般够你识别几百页古籍。
</details>
